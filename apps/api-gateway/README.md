## Installation

```
yarn install
```
## Setup

Create .env file and set parameters

```
AUTH_SERVICE_RMQ_URL=
AUTH_SERVICE_RMQ_QUEUE=

USER_SERVICE_RMQ_URL=
USER_SERVICE_RMQ_QUEUE=

WALLET_SERVICE_RMQ_URL=
WALLET_SERVICE_RMQ_QUEUE=

REDIS_HOST=
REDIS_PORT=
REDIS_MAX=
REDIS_TTL=

ADMIN_API_KEY=
```

## Running the app

```
yarn start
```

## Test

```
yarn test
```
