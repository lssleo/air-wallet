## Installation

```
yarn install
```
## Setup
Generate Prisma client:
```
prisma generate
```

Create .env file and set parameters:

```
DATABASE_URL=

USER_SERVICE_RMQ_URL=
USER_SERVICE_RMQ_QUEUE=

AUTH_SERVICE_RMQ_URL=
AUTH_SERVICE_RMQ_QUEUE=

EXPIRATION=

JWT_SECRET=
```

## Running the app

```
yarn start
```

## Test

```
yarn test
```
